## Feature Locations

### Where to Find Key Features

#### Events Management (Admin Only)
- **Location**: Admin Navigation → Events
- **Actions**: Create, Edit, Delete events
- **Features**: Date/time picker, location, attendee tracking

#### Analytics (Admin Only)
- **Location**: Admin Navigation → Analytics
- **Charts**: Pie, Bar, Line, Area charts
- **Metrics**: User distribution, engagement, performance

#### Mentorship Program (Student Only)
- **Location**: Student Navigation → Mentorship
- **Actions**: Browse alumni, send requests, track status
- **Features**: Search, filter by department, status tracking

#### Games (Alumni & Students)
- **Location**: Navigation → Games
- **Games**: 8 Queens Puzzle, Mini Sudoku
- **Features**: Leaderboards, score tracking, timers

#### AI Helper (All Authenticated Users)
- **Location**: Floating icon in bottom-right corner
- **Click**: Opens chat sidebar
- **Features**: Job suggestions, FAQ answers, newsletter summaries

## Page Interactions

### Dashboard Pages

**Admin Dashboard:**
- View statistics in 4 colorful cards
- Interact with charts (hover for details)
- Scroll through recent activity feed

**Alumni Dashboard:**
- Click on job cards to see details
- View upcoming events
- See recent posts from community

**Student Dashboard:**
- Click "Connect" on alumni cards to request mentorship
- Browse featured job opportunities
- Track your mentorship request status

### Events Management (Admin)

**Creating an Event:**
1. Click "Create Event" button
2. Fill in the form:
   - Title
   - Description
   - Date & Time
   - Location
3. Click "Create Event" to save

**Editing an Event:**
1. Find the event in the list
2. Click "Edit" button
3. Modify details in modal
4. Click "Update Event"

**Deleting an Event:**
1. Click trash icon on event
2. Confirm deletion

### Mentorship Flow (Student)

**Requesting a Mentor:**
1. Navigate to Mentorship page
2. Browse alumni or use search/filter
3. Click "Request Mentorship" on desired alumni
4. Write a personalized message
5. Click "Send Request"

**Tracking Requests:**
- View all requests at top of Mentorship page
- Status shown with colored badges:
  - Yellow: Pending
  - Green: Accepted
  - Red: Declined

### Games

**Playing 8 Queens:**
1. Go to Games → Play 8 Queens
2. Click squares to place/remove queens
3. Green = valid placement
4. Red = invalid placement
5. Complete puzzle when all 8 queens placed correctly
6. Score saved automatically

**Playing Mini Sudoku:**
1. Go to Games → Play Mini Sudoku
2. Click empty cell
3. Select number 1-4
4. Red cells = conflicts
5. Timer runs automatically
6. Complete puzzle to save score

### AI Helper

**Using the AI Assistant:**
1. Click floating icon in bottom-right
2. Type your question
3. AI responds instantly
4. Ask about:
   - Jobs ("Show me job opportunities")
   - Newsletters ("Summarize latest newsletter")
   - Events ("What events are coming up?")
   - General help ("How do I use this platform?")

## Common Workflows

### Admin: Creating and Managing an Event
```
Dashboard → Events → Create Event → Fill Form → Save
         → View in Upcoming Events → Edit if needed
```

### Student: Finding a Mentor
```
Dashboard → Mentorship → Search/Filter → View Profile
         → Request Mentorship → Write Message → Send
         → Track Status in Your Requests
```

### Alumni: Posting a Job
```
Dashboard → Jobs → (Future: Add Job Button)
         → Fill Job Details → Post
         → Students can now see it
```

### Anyone: Playing a Game
```
Games Menu → Select Game (8 Queens or Sudoku)
          → Play and Complete → Score Saved
          → View Your Rank in Leaderboard
```
